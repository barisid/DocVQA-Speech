# DocVQA-Speech

Document QA System with Speech Interface
An intelligent document question-answering system that combines speech recognition, visual document understanding, and text-to-speech capabilities. Ask questions about your documents using your voice and get spoken answers!

Features

Speech-to-Text: Record questions using your microphone (powered by OpenAI Whisper)
Document Visual QA: Analyze and understand document content (powered by Donut DocVQA)
Text-to-Speech: Listen to answers spoken aloud (powered by pyttsx3)
Dual Input Modes: Choose between voice recording or text typing
Visual Understanding: Works with document images including invoices, forms, receipts, and more
